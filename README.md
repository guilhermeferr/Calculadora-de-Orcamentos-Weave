# 🚀 Weave - Sistema de Orçamentos (V21)

Uma ferramenta leve e eficiente para calcular orçamentos da **Weave**, otimizada para lidar com grandes volumes de dados (mais de 18.000 itens) com performance instantânea.

## 🛠️ O que este projeto faz?
Este projeto foi desenvolvido para a **Weave** resolver o problema de busca lenta e travamentos ao processar tabelas gigantescas. Ele utiliza uma base de dados em CSV para garantir que a pesquisa de produtos como **Cartão de Visita 300g** ou **Banners** seja feita em milissegundos, agilizando o atendimento ao cliente.

## ✨ Funcionalidades
- **Identidade Weave:** Orçamentos formatados prontos para envio via WhatsApp com a marca da empresa.
- **Busca Inteligente:** Filtros otimizados que priorizam automaticamente cartões e materiais de comunicação visual.
- **Cálculo Automático:** Aplica a margem de lucro de 30% da Weave sobre o custo, frete e arte.
- **Logística Integrada:** Frete mínimo já configurado em R$ 13,00 para agilizar o cálculo.
- **Totalmente Local:** Funciona direto no navegador, garantindo privacidade e velocidade sem depender de internet após carregar a base.

## 🚀 Como usar
1. Baixe o repositório ou acesse o link do projeto via GitHub Pages.
2. Selecione o arquivo `tabela_zap.csv` (sua base de dados atualizada) no botão de importação.
3. Digite o produto desejado (ex: "cartao 300" ou "banner").
4. Clique em copiar e envie o orçamento profissional da Weave para o seu cliente.
